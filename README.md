-   nvm with [shell integration](https://github.com/nvm-sh/nvm#bash)
-   Node [`lts/gallium`](https://nodejs.dev/en/about/releases/)
-   NPM 8

2. `npm run fractal:start` (Runs Gulp and starts the Fractal preview server)
